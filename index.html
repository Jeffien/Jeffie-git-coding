Git commmands

    Command list:

  • git add . : Lägger till ändringar
  • git push : laddar upp till git kontot
  • git commit -m "kommentar": beskriver de ändringar som gjorts
  • git status: status för git
  • git init: skapar en git fil av redan existerande filer
  • git pull: uppdaterar filen från github
  • git log: visar historiken för git
